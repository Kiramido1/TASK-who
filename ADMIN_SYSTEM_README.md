### 🚀 كيفية الاستخدام

#### للمستخدمين العاديين:
1. افتح الموقع وادخل اسمك
2. هتشوف الـ Admins المتاحين (Fahmy و Ewis)
3. اضغط على "طلب مشاركة الشاشة" للـ Admin اللي عايزه
4. هتدخل في طابور الانتظار
5. هتشوف موقعك في الطابور وعدد المنتظرين
6. لما الـ Admin يقبلك، هتبدأ الجلسة تلقائياً
7. بعد ما تخلص، الـ Admin هينهي الجلسة واللي بعدك هيدخل تلقائياً

#### للـ Admins:
1. اضغط على زر "وضع الإدارة" في أعلى الشاشة
2. سجل دخول باسم المستخدم وكلمة المرور
3. هتشوف لوحة التحكم فيها:
   - عدد المنتظرين
   - حالتك (متاح/مشغول)
   - الجلسة الحالية
   - قائمة الانتظار الكاملة
4. اضغط "قبول" لبدء جلسة مع مستخدم
5. اضغط "إنهاء الجلسة" لما تخلص
6. المستخدم التالي هيدخل تلقائياً

### 📊 المعلومات المعروضة

#### للمستخدمين:
- عدد المنتظرين الكلي
- موقعك في الطابور
- حالة كل Admin (متاح/مشغول)
- مين اللي في جلسة مع كل Admin
- حالة جلستك (نشطة/في الانتظار)

#### للـ Admins:
- عدد المنتظرين
- حالتك الحالية
- معلومات الجلسة النشطة
- قائمة كاملة بالمنتظرين مع:
  - الترتيب
  - الاسم
  - وقت الانتظار
  - أزرار القبول/الرفض

### 🎨 التصميم

- تصميم احترافي بنفس ستايل الموقع الأصلي
- Glass effect و animations
- Gold gradient للعناصر المهمة
- Responsive على كل الشاشات
- Real-time updates باستخدام Firebase

### 🔄 الـ Queue System

- نظام طوابير تلقائي
- ترتيب حسب وقت الطلب (First Come First Served)
- انتقال تلقائي للمستخدم التالي بعد انتهاء الجلسة
- إمكانية الإلغاء في أي وقت
- تحديث فوري لكل المستخدمين

### 🔐 الأمان

- كلمات مرور قوية للـ Admins
- Session management آمن
- تخزين الـ session في localStorage
- حماية من الوصول غير المصرح
